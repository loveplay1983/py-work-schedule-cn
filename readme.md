## 如何运行

### 先决条件
安装所需的库：
```bash
pip install flask openpyxl pyside6
```

### Web 应用程序
1. 将 `common.py` 放在与 `web_app/` 相同的目录中。
2. 导航到 `web_app/` 并运行：
```bash
python app.py
```
3. 打开浏览器访问 `http://127.0.0.1:5000/`。

### 桌面应用程序
1. 将 `common.py` 放在与 `desktop_app/` 相同的目录中。
2. 导航到 `desktop_app/` 并运行：
```bash
python main.py
```
